Installing
==========

GitHub
------

.. code-block:: bash

    pip install git+http://github.com/ITCase/ps_tree.git

PyPi
----

.. code-block:: bash

    pip install ps_tree

Source
------

.. code-block:: bash

    git clone git+http://github.com/ITCase/ps_tree.git
    python setup.py install

Contribute
----------

.. code-block:: bash

    git clone git+http://github.com/ITCase/ps_tree.git
    python setup.py develop
